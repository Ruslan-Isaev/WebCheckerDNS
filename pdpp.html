DNS Checker

Политика обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных описывает, как и с какой целью обрабатываются персональные данные пользователей сайта REDACTED.

2. Какие данные собираются

Мы не собираем и не храним персональные данные пользователей. Единственные данные, которые обрабатываются — это доменные имена, введённые в форму для проверки DNS-записей.

3. Цели обработки данных

Введённые домены используются исключительно для отправки запросов к сторонним DNS-сервисам с целью предоставления результатов пользователю.

4. Передача данных третьим лицам

Данные доменов передаются только сторонним DNS-сервисам (Google DNS, Cloudflare, AliDNS) в целях выполнения запросов и не хранятся на наших серверах.

5. Безопасность данных

Мы предпринимаем все необходимые меры для защиты данных пользователей и используем только надёжные сторонние сервисы.

6. Изменения в политике

Мы можем обновлять данную политику по мере необходимости. Рекомендуется периодически проверять актуальность информации.
